## 📖 How to Use These Notes

1. **Start with Module 1** if you're new to Git
2. **Follow the modules in order** — each builds on the previous one
3. **Use the Cheat Sheet** (Module 7) as a quick reference while working
4. **Open in VS Code** and use `Ctrl+Shift+V` (or `Cmd+Shift+V` on Mac) to preview markdown with images
5. **Practice every command** in a real terminal — reading alone isn't enough!
6. **Export to PDF** — Use VS Code extension "Markdown PDF" or paste into Notion/Google Docs for a formatted PDF
